# UE Web Showcase

Lean Next.js site (no CMS) to introduce our team and host a UE game stream later.

## Scripts
- dev: local dev server
- build / start: production build & run
- lint / type-check: quality gates

## Env
Copy `.env.local.example` to `.env.local` when integrating streaming.

## Structure
- `/app/(site)` public pages
- `/app/(system)` robots/sitemap/privacy/terms
- `/components`, `/lib`, `/styles`, `/config`
